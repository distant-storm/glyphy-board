# Open-Meteo Weather API Configuration
# 
# The weather_info.py script now uses Open-Meteo API instead of Met Office
# 
# Benefits of Open-Meteo:
# - Free to use, no API key required
# - High accuracy weather data
# - Global coverage
# - Reliable and fast API
# - Based on multiple weather models
#
# To use the weather_info.py script:
# 1. No setup required - just run the script!
# 2. Run: python3 scripts/weather_info.py
# 3. Weather data will be saved to: data/weather_data.json
#
# Open-Meteo API: https://open-meteo.com/
# Documentation: https://open-meteo.com/en/docs
